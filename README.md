## Development

### Module Compilation

To compile the Scry module after making changes:

```bash
npm run bmad:compile
```

This processes all entities, agents, and workflows in `.bmad/scry/` and makes them available as commands.

The project integrates with multiple AI coding assistants:
- **Claude Code** - Primary development environment
