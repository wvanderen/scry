# Session Notes

This directory can hold notes from specific ritual sessions, divination readings, or significant spiritual work.

## Suggested Structure

Create files as needed for:
- Ritual transcripts
- Divination readings with interpretations
- Dream journals
- Shadow work sessions
- Significant breakthroughs or visions

## Naming Convention

Suggested format: `YYYY-MM-DD-session-type.md`

Examples:
- `2025-01-15-invocation.md`
- `2025-01-20-tarot-reading.md`
- `2025-02-01-shadow-work.md`
